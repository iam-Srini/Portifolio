**Project 1: Data Migration from Google Cloud Storage to Snowflake with Power BI Visualization**

*Objective:*
This project focuses on creating a simple data workflow to move data from Google Cloud Storage (GCS) to Snowflake for processing and then visualizing the results using Power BI. The goal is to demonstrate the ability to handle data ingestion, transformation, and visualization for analytical purposes.

*Technologies Used:*
Google Cloud Storage (GCS): Used for storing raw data files.
Snowflake: Used for data processing and transformation using SQL.
Power BI: Used for data visualization, creating interactive reports and dashboards.

*Workflow:*
Data Ingestion:
Data is manually extracted from GCS using Snowflake connectors.

Data Transformation:
The data is loaded into Snowflake where transformations are applied using SQL scripts.
The transformations ensure that the data is structured and cleaned for analysis.

Data Visualization:
Power BI is connected to Snowflake to fetch the processed data and create visualizations.
Dashboards are created to provide insights based on the transformed data.
